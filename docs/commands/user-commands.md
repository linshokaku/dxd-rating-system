# ユーザー向けコマンド

## `/register`

### 目的
実行者をマッチングシステムのプレイヤーとして登録する。

### 実行者
サーバー内の全ての Discord ユーザー。

### 入力
- なし。

### 正常時の挙動
- 実行したユーザーの Discord user ID をマッチングシステムに登録する。
- レスポンスを返す
  - `登録が完了しました。`

### エラー時の挙動
- すでに登録済みの場合:
  - `すでに登録済みです。`
- 内部エラーが発生した場合:
  - `登録に失敗しました。管理者に確認してください。`
  - 内部例外の詳細は Discord に出さない。

### 権限
- 特になし。
