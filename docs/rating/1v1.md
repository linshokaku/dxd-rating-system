# 1v1 レート計算仕様

## 目的

`1v1` 対戦に対して、各プレイヤーの独立した `1v1` レートを更新する。

## 期待勝率

Team A のプレイヤーのレートを `r_A`、Team B のプレイヤーのレートを `r_B` とする。

```text
q_A = 10^(r_A / 400)
q_B = 10^(r_B / 400)
```

```text
p_A = q_A / (q_A + q_B)
p_B = q_B / (q_A + q_B)
```

## レート更新式

`1v1` の人数係数は `1` なので、実効 K は `K_A_eff = K_A * 1`、`K_B_eff = K_B * 1` となる。

Team A:

```text
r_A' = r_A + K_A_eff * (y - p_A)
```

Team B:

```text
r_B' = r_B - K_B_eff * (y - p_A)
```

## マッチ処理手順

1. 参加 2 人の `1v1` 用 `player_format_stats` を取得する
2. `q_A`, `q_B`, `p_A` を計算する
3. 実結果 `y` を決める
4. 各プレイヤーの K を、その `1v1.games_played` から決める
5. 新しい `rating` を計算する
6. 勝ち、負け、引き分けの場合のみ `games_played`, `wins`, `losses`, `draws` を更新する

## 無効マッチ

無効マッチでは、`1v1` レートと `1v1` 戦績を更新しない。
