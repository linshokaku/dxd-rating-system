# 2v2 レート計算仕様

## 目的

`2v2` 対戦に対して、各プレイヤーの独立した `2v2` レートを更新する。

## 期待勝率

各プレイヤーのレート `r` から強さ指標 `q` を計算する。

```text
q_i = 10^(r_i / 400)
```

Team A, Team B のチーム強さを以下で求める。

```text
Q_A = q_A1 + q_A2
Q_B = q_B1 + q_B2
```

```text
p_A = Q_A / (Q_A + Q_B)
p_B = Q_B / (Q_A + Q_B)
```

## レート更新式

Team A のプレイヤー `i`:

```text
r_i' = r_i + K_i * (q_i / Q_A) * (y - p_A)
```

Team B のプレイヤー `j`:

```text
r_j' = r_j - K_j * (q_j / Q_B) * (y - p_A)
```

## 試合処理手順

1. 参加 4 人の `2v2` 用 `player_format_stats` を取得する
2. 各プレイヤーの `q` を計算する
3. `Q_A`, `Q_B`, `p_A` を計算する
4. 実結果 `y` を決める
5. 各プレイヤーの K を、その `2v2.games_played` から決める
6. 新しい `rating` を計算する
7. 勝ち、負け、引き分けの場合のみ `games_played`, `wins`, `losses`, `draws` を更新する

## 無効試合

無効試合では、`2v2` レートと `2v2` 戦績を更新しない。
