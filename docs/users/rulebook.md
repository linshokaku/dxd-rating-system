# ユーザー向けルールブック

## この文書について

- この文書は、プレイヤーがレート戦へ参加するための基本ルールをまとめた案内です。
- 現在の対象ゲームは `Drag x Drive` です。
- 通常案内は `3v3` を想定していますが、`1v1` と `2v2` を運用する場合も、特記がない限り同じ考え方を適用します。
- 緊急時の運営案内や admin の個別判断がある場合は、その案内を優先します。

## 基本姿勢

- マッチは公正な対戦と円滑な進行を前提としています。
- レーティングシステムと admin の案内、決定したチーム分け、対象フォーマットに従ってください。
- `Drag x Drive` のプレイヤー人口の増加の障害となるような行動には厳正に対処します。

## マッチングの進行

### 1. キュー参加

- 「レート戦マッチングチャンネル」からキューへ参加することができます。
- 自分のレートから参加可能な「試合形式」「階級」から好きなキューを選択することができます。
- キュー参加後の有効時間は 30 分です。
- 一度に参加可能なキューは一つだけです。
- マッチングへの参加が困難になった場合は、参加中のキューの「マッチングキャンセル」ボタンよりキャンセルしてください。
- 他のマッチングキューへ参加したくなった場合は、一度、現在参加中のキューの「マッチングキャンセル」ボタンからキャンセルしてから、他のマッチングキューへ参加してください。

### 2. マッチ成立

- 必要人数がそろうとマッチが成立します。
- マッチ成立後は、参加者向けのマッチ運営用スレッドにチーム分けと進行案内が送られます。
- 以後の連絡、勝敗報告、承認は原則としてそのスレッドで行います。

### 3. 親の決定

- マッチ成立後、最初に進行役となる親を決めます。
- 親募集期間は 5 分です。
- 立候補がなければ、Bot が参加者の中からランダムに親を決定します。
- 親は部屋立てと進行をする役割ですが、問題が起きた場合は抱え込まず admin へ連絡してください。
- マッチ運営を円滑に進める目的で、必要に応じて親以外が部屋を立てたり、進行することができます。

### 4. 対戦の実施

- マッチは基本的に 3 試合制で、先に 2 勝したチームが勝ちです。
- 部屋立て、試合開始、試合間の準備は参加者と親で進行します。
- 親はパークIDで参加可能なフレンドパークを新しく作成し、パークIDをマッチ運営用スレッドに共有してください。
- パークIDの共有は、親になった、もしくは選ばれてから 5 分以内に行われる必要があります。そうしなかった場合、「部屋立て遅延」ペナルティが与えられます。部屋立て遅延があった場合はマッチ参加者のいずれかがマッチ運営用スレッド内で admin へのメンションを含めて該当者を報告してください。
- マッチ参加者は親が試合を正しくセットできるように、ゲーム内のプレイヤー名をマッチ運営用スレッドで共有してください。
- 不特定多数のマッチ参加者、観戦希望者が円滑にパークに参加できるようにするために、フレンド合流可能なフレンドパークでのマッチ運営はなるべく避けるようにしてください。フレンド合流可能なフレンドパークでのマッチ運営は現状では禁止、違反行為としては扱いません。ただし、admin の判断で「基本姿勢」に記載されている内容に抵触していると判断された場合はその限りではありません。
- 試合参加者は、パークIDが共有されてから 5 分以内にパークに参加してください。そうしない場合、「遅刻」ペナルティが与えられます。遅刻があった場合はマッチ参加者のいずれかがマッチ運営用スレッド内で admin へのメンションを含めて該当者を報告してください。ゲーム内のメニューの「メンバー」タブの内容が証拠となるので、保存してください。該当者からの異議申し立てがあった場合には証拠の提出が必要となります。証拠の提出には該当スクリーンショットと撮影時刻を合わせたものが必要となるので、「Nintendo Switch App」より該当スクリーンショットの撮影した日時が含まれる画面のスクリーンショットを用意してください。そのような場合に証拠の提出が行われなかった場合は、報告した人にペナルティが付与される場合があります。
- 一人以上の遅刻者がいた場合、そのマッチを「無効マッチ」として申請してください。
- マッチ参加者全員の間で合意が取れる場合は、状況に応じてマッチの開始を遅らせることができます。

### 5. マッチ進行

- マッチ進行は親によって行われます。
- マッチは 3 試合、もしくは一方のチームが 2 勝するまで行います。
- 親はレーティングシステムによって案内されたチーム分けに従って、参加メンバーを振り分けて試合を開始してください。
- 試合メンバーの初期配置は親が自由に決めることができます。ただし、`1v1` 形式と `2v2` 形式において、意図的に不公平な初期配置を繰り返した場合は、「マッチ進行ミス」ペナルティが付与される場合があります。故意性があると思われる場合は、マッチ運営用スレッド内で admin へのメンション付きで報告してください。報告する際は可能な限り証拠を提示してください。
- 試合の開始は、マッチ参加者が揃う、もしくは、直前の試合が終了してから 3 分以内に始める必要があります。3 分を超えて開始されない場合は、「マッチ進行ミス」ペナルティが親に付与されます。マッチ参加者のいずれかが、マッチ運営用スレッドで admin へのメンションを含めて該当者を報告してください。可能な範囲で証拠として主張可能なスクリーンショットまたは動画を提出してください。
- 親は上記に違反しない範囲で好きなタイミングで試合を開始することができます。ただし、可能な限りマッチ参加者の準備が整っていることを確認し、不必要に遅延させずに開始することが推奨されます。意図的にマッチ参加者の準備が整っていない状態での試合開始や上記に違反しない範囲での試合開始の遅延は望ましいことではないですが、違反対象ではありません。ただし、admin の判断で「基本姿勢」に記載されている内容に抵触していると判断された場合はその限りではありません。
- マッチ参加者は、試合前、試合間にキャラの変更を行ってもいいです。
- 試合メンバーのチーム分けを間違えた場合は、プレイ中の試合の終了をもって、直ちにマッチを終了し、そのマッチを「無効マッチ」として申請してください。その場合、親には「マッチ進行ミス」ペナルティが付与されます。マッチ参加者のいずれかが、マッチ運営用スレッドで admin へのメンションを含めて該当者を報告してください。試合終了時点でのリザルト画面が証拠となるので、保存してください。該当者から異議申し立てがあった場合には証拠の提出が必要となります。そのような場合に証拠の提出が行われなかった場合は、報告した人にペナルティが付与される場合があります。
- マッチ進行ミスがあった場合でもマッチ参加者全員の間で合意が取れる場合は、状況に応じてマッチの進行を継続することができます。
- マッチ進行中に、パークからの離脱、もしくは回線の切断があった場合は、プレイ中の試合の終了をもって、直ちにマッチを終了し、そのマッチを「無効マッチ」として申請してください。その場合、切断したプレイヤーには「切断」ペナルティが付与されます。マッチ参加者のいずれかが、マッチ運営用スレッドで admin へのメンションを含めて該当者を報告してください。試合終了時点でのリザルト画面が証拠となるので、保存してください。該当者から異議申し立てがあった場合には証拠の提出が必要となります。そのような場合に証拠の提出が行われなかった場合は、報告した人にペナルティが付与される場合があります。
- 切断があった場合でもマッチ参加者全員の間で該当マッチの勝敗の扱いを含む合意が取れる場合は、状況に応じてマッチの進行を継続することができます。

### 6. 勝敗報告

- 3 試合終了、もしくは一方のチームが 2 勝した時点で、マッチを終了としてください。
- マッチ終了後は、参加者全員が勝敗報告を行ってください。
- 報告できるのは、そのマッチの参加者本人だけです。
- 報告は自分視点で `勝ち`、`負け`、`引き分け` から選んでください。
- `勝ち`、`負け`、`引き分け` の受付開始は親決定から 7 分後です。
- マッチが正常に終了できなかった場合は、「無効マッチ申請」をしてください。
- 通常の勝敗報告の締切は親決定から 27 分後です。
- 勝敗報告を忘れた場合、「勝敗無報告」ペナルティが与えられます。
- 勝敗報告受付中は、何度でも自分の最新報告を上書きできます。

### 7. 承認

- 全員の報告が一致した場合は、そのまま即時に結果が確定します。
- 報告が食い違った場合や未報告者がいる場合は、承認期間に入ります。
- 承認期間に入る場合、マッチの結果がレーティングシステムによって仮決定されます。
- マッチの結果の仮決定は次のルールによって行われます。
  - 全てのマッチの勝敗報告から、最も多数となる勝敗結果が一つに定まる場合、その結果で勝敗を仮決定します。
  - 同票数となる結果のうち、親の投票が含まれている結果があれば、その結果で勝敗を仮決定します。
  - どれにも当てはまらない場合は、「無効マッチ」として仮決定します。
- 承認期間は、全員の報告の完了、もしくは、勝敗報告の締切時刻から 5 分です。
- 自分が承認対象になった場合は、案内に従って時間内に承認してください。
- 仮決定したマッチ結果が異なっている場合は承認せず、admin へ連絡してください。
- 勝敗無報告のマッチ参加者が仮決定結果を承認しなかった場合、「勝敗無報告」ペナルティが与えられます。
- 仮決定した結果と食い違う報告をしたマッチ参加者が仮決定結果を承認しなかった場合、「勝敗誤報告」ペナルティが与えられますが、証拠を admin に提示し、マッチ結果が修正される場合は、ペナルティの取り消しが行われます。
- 試合結果の修正を admin に求める場合は、証拠を用意することが推奨されます。毎試合の試合結果のスクリーンショットが最適です。

### 8. 結果確定

- 結果が確定した後は、参加者による勝敗報告の変更や承認はできません。
- 確定した結果に応じて、レーティングや戦績が更新されます。
- 必要に応じて admin が結果を修正する場合があります。

## 無効マッチの扱い

- 通常の勝敗として確定させるべきでない場合は、`無効マッチ` を選んでください。
- `無効マッチ` の申請はマッチ成立直後から行えます。
- 判断が割れる場合は、マッチ運営用スレッド内で admin へメンションを含めて連絡してください。
- 無効マッチになった対戦では、原則としてレーティングや戦績は更新されません。

## 違反行為

- 勝敗誤報告
  - 確定結果と異なる内容を最終報告することです。
  - 仮決定結果を承認している場合には適用されません。
- 勝敗無報告
  - 必要な期限までに報告せず、承認も行わないことです。
- 部屋立て遅延
  - 親になった後、5 分以内にパークIDをマッチ運営用スレッドに共有しないことです。
- マッチミス
  - 対象メンバー、ルール、対戦相手の取り違えなどにより、正しい対戦として扱えない状態にすることです。
- 遅刻
  - パークID共有後 5 分以内に該当パークに参加しないことです。
- 切断
  - マッチ運営中に該当パークから抜ける、もしくは、試合中に回線の切断を行うことです。
- その他の進行妨害や公正性を著しく損なう行為
  - 事実と異なる説明、意図的な混乱の誘発、運営判断の妨害などを含みます。
  - 暴言、レーティングシステム参加者を不快にさせるような言動を含みます。
  - 「基本姿勢」に抵触するような行為を含みます。

## ペナルティと運営対応

- `勝敗誤報告` は、自動ペナルティとして `+1` が付与されます。
- `勝敗無報告` は、自動ペナルティとして `+1` が付与されます。
- ただし、承認対象者が承認期間内に承認した場合は、自動ペナルティ対象から外れます。
- `部屋立て遅延`、`マッチミス`、`遅刻`、`切断` などは、admin が状況を確認して手動で対応します。
- 違反の内容や頻度によっては、キュー参加制限などの追加対応が行われる場合があります。

## マッチ観戦

- 「レート戦マッチ速報」チャンネルで成立したマッチの「観戦」ボタンからマッチ運営用スレッドに参加することができます。
- 観戦者はマッチ運営用スレッドに共有されているパークIDから、マッチが運営されているフレンドパークに合流することができます。
- フレンド合流可能なフレンドパークでマッチ運営が行われており、マッチ参加者にフレンドがいる場合は、フレンド合流することもできます。
- マッチ運営の障害とならないように、マッチ参加者のプレイヤー名と似た名前での合流は避けるようにしてください。
- その他、ゲーム内でのマッチ運営の妨げとなるような行動や混乱を招く行動は避けるようにしてください。
- マッチ運営の障害とならない範囲でマッチ運営用スレッド内でのコミュニケーションを取ることができます。
- マッチ運営の障害となる行為を繰り返している、もしくは「基本姿勢」の内容に抵触していると admin によって判断された場合は、観戦の制限などの追加対応が行われる場合があります。

## 困ったときの対応

- マッチを正常に続けられないと感じたら、できるだけ早くマッチ運営用スレッド内で admin へのメンション付きで共有してください。
- 事実関係が争点になる場合は、証拠の提示が必要になる場合があります。スクリーンショットなどの証拠を適宜保存しておいてください。
- 自分だけで解決できない場合は、admin へのメンションもしくはDMを送り相談してください。
